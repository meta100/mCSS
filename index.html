mCSS an extension to jQuery.css()

requires jQuery 1.4.3 or higher.
mCSS extends the jQuery.css() adding the following calls (without changing the original use):

  • $('foo').css('property', 'value') - same result as before this extension.
  • $('foo').css({'property': 'value'}) - same result as before this extension.
  • $('foo').css('property', 'value', 'inline') - same result as before this extension.
  • $('foo').css({'property': 'value'}, 'inline') - same result as before this extension.
  • $('foo').css('property') - same result as before this extension.
  • $('foo').css('property', 'value', 'low') - adds css to a style sheet prepended to the <head>.
  • $('foo').css({'property': 'value'}, 'low') - adds css to a style sheet prepended to the <head>.
  • $('foo').css('property', 'value', 'high') - adds css to a style sheet appended to the <body>.
  • $('foo').css({'property': 'value'}, 'high') - adds css to a style sheet appended to the <body>.
  • $('foo').css() - same result as before this extension.
  • $('foo').cssCopyTo('bar') - copies the diff css from $('foo') to $('bar') and applies to style attribute - 'bar' can be a selector or an object.
  • $('foo').cssCopyTo('bar', 'inline') - copies the diff css from $('foo') to $('bar') and applies to style attribute.
  • $('foo').cssCopyTo('bar', 'low') - copies the diff css from $('foo') to $('bar') and applies to a style sheet prepended to the <head>.
  • $('foo').cssCopyTo('bar', 'high') - copies the diff css from $('foo') to $('bar') and applies to a style sheet appended to the <body>.
  • $('foo').cssSyncTo('bar') - will apply any $.css changes made to 'foo' also to 'bar.

mCSS uses csshooks

Example 1: Border Radius

#example1 { border-radius: 10px; }

Example 2: Box Shadow

#example2 { box-shadow: 10px 10px 5px #888; }

Example 3: Rotate

#example3 { transform: rotate(-15deg); }

Example 4: Rotate

#example4 { opacity: .5; }
